## Rules for GitHub Topics

- ✅ Maximum 20 topics allowed
- ✅ Use lowercase letters, numbers, and hyphens only
- ✅ 50 characters or less per topic
- ✅ Topics are exact match only (no partial matching)
- ✅ More specific topics = better discoverability

1. Your repository will appear in topic searches: `https://github.com/topics/rvtools`

4. Topics help GitHub's recommendation algorithm
