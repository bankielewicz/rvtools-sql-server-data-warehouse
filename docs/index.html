RVTools SQL Server Data Warehouse

Import, warehouse, and report on VMware inventory exports with complete historical tracking

Features

  • Full RVTools Support - Import all 27 RVTools tabs (~850 columns)
  • SCD Type 2 History - Track all configuration changes with ValidFrom/ValidTo timestamps
  • 24 Pre-built Reports - Inventory, Health, Capacity, and Trends analysis via SSRS or Web interface
  • Web Dashboard - Modern ASP.NET Core 8.0 browser-based reporting with interactive charts
  • PowerShell ETL Pipeline - Automated import with comprehensive error handling
  • Audit Trail - Complete logging of all imports and data changes
  • Multi-vCenter Support - Consolidate data from multiple VMware environments
SQL Server 2016+
PowerShell 5.1+
ASP.NET Core 8.0
SSRS Reports
